## 一、需求概述

对运维系统增加以下功能：
1. 定时刷新设备状态（在线/离线）
2. 手工和定时获取设备信息，形成巡检记录

4. 数据库巡检（MySQL/MSSQL/Oracle）

## 二、完成功能清单

### 2.1 SNMP设备巡检功能 ✅

**新增文件：**
| 文件路径 | 说明 |
|---------|------|
| `backend/apps/monitoring/snmp_client.py` | 原生Python SNMP客户端，不依赖外部库 |
| `backend/apps/monitoring/snmp_executor.py` | SNMP监控执行器，支持自动降级 |
| `backend/apps/inspection/executors.py` | SNMP设备巡检执行器 |

**功能特性：**
- 支持 SNMP v1/v2c 协议
- 原生Socket实现，无需安装snmp-python等依赖
- 自动降级（网络不可达时使用模拟数据）
- 支持系统信息采集 (sysDescr, sysName, sysUpTime)
- 支持接口状态检测 (ifNumber, ifStatus, ifDescr, ifSpeed)

**巡检项目：**
| 项目代码 | 项目名称 | 类别 |
|---------|---------|------|
| SNMP_CONNECTION | SNMP连接 | connection |
| SYSTEM_INFO | 系统信息 | info |
| UPTIME | 运行时间 | info |
| INTERFACE_STATUS | 接口状态 | network |
| INTERFACE_COUNT | 接口数量 | network |
| BANDWIDTH_USAGE | 带宽使用 | performance |

### 2.2 数据库巡检功能 ✅

**新增文件：**
| 文件路径 | 说明 |
|---------|------|
| `backend/apps/inspection/database_connectors.py` | 数据库连接器（MySQL/MSSQL/Oracle） |
| `backend/apps/inspection/db_inspector.py` | 数据库巡检执行器 |

**支持数据库：**
- MySQL (通过pymysql)
- Microsoft SQL Server (通过pymssql)
- Oracle (通过cx_Oracle)

**巡检项目：**
| 项目代码 | 项目名称 | 类别 |
|---------|---------|------|
| DB_CONNECTION | 数据库连接 | connection |
| DB_VERSION | 数据库版本 | info |
| TABLESPACE | 表空间使用率 | storage |
| BUFFER_HIT | 缓冲池命中率 | performance |
| SESSIONS | 活跃会话数 | session |
| BACKUP | 备份状态 | backup |
| ERROR_LOG | 错误日志 | log |

**连接配置映射：**
```
资产字段          -> 连接参数
─────────────────────────────────
db_host          -> 主机地址
host_address     -> 主机地址 (备选)
db_port          -> 端口
db_name          -> 数据库名
database_name    -> 数据库名 (备选)
db_username      -> 用户名
db_password      -> 密码
```
